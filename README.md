# Traductor de Gestos
Bienvenidos, este proyecto busca traducir gestos especificos que realizamos con nuestras manos.

## Funcionamiento
Dado un gesto realizado con las manos, la app detecta el patrón específico y lo traduce; en formato de texto que se despliega en la pantalla, y en formato de sonido a través de la salida de audio.

Los gestos incluidos hasta el momento son los números del 1 al 5

## Lanzar aplicación
Para poder ejecutar la app es necesario seguir la siguiente secuencia de comandos en bash:
1. **`> python3 -m venv mp_env`**: Creamos el entorno virtual sobre el cual la app va a funcionar
2. **`> source mp_env/bin/activate`**: Activamos el entorno virtual
3. **`> pip install -r requirements.txt`**: Instalación de las dependencias necesarias para que la app funcione
